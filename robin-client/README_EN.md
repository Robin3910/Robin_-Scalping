# Robin Scalper - Tauri Client

A grid trading client for Binance futures, built with Tauri + React + TypeScript.

## Features

- Grid trading strategy (Long/Short)
- Real-time K-line charts
- Technical indicators (EMA, RSI, MACD, ADX)
- Paper trading / Live trading
- Binance Testnet support
- Trailing stop / Partial TP / Breakeven
- Daily open limit / Cooldown after close

## Quick Start

```bash
cd robin-client
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri build
```
